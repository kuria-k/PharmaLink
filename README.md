# PharmaLink POS System

PharmaLink is a modern Point of Sale (POS) system designed for pharmacies and retail businesses.  
It streamlines operations by integrating **Admin**, **Sales**, **Cashier**, and **Inventory** modules into one unified platform.

---

## Features

- **Admin Module**
  - Manage user accounts and roles
  - Configure system settings
  - Generate detailed reports and analytics

- **Sales Module**
  - Process customer purchases efficiently
  - Apply discounts, VAT, and print invoices
  - Track daily, weekly, and monthly sales performance

- **Cashier Module**
  - Easy-to-use interface for quick transactions
  - Secure payment handling (cash, card, mobile money)
  

- **Inventory Module**
  - Real-time stock management
  - Track product batches and expiry dates
  - Automated low-stock alerts and restocking suggestions

---

## Tech Stack

- Backend: (DJANGO)
- Frontend: (REACT)
- Database: (SQLITE)
- Deployment: (RENDER)
